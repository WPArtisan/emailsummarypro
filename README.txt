== Installation ==

1. Upload `email-summary-pro` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will start receiving emails automatically once a week.
4. Visit the 'Email Summary Pro' submenu item under Options for more settings.
